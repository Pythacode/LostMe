Collecte de données personnelles

Dans le cadre du fonctionnement de Lostme, les données suivantes sont collectées à chaque envoi de message :
  • Le nom saisi par l'utilisateur
  • L'adresse IP
  • Le user agent (navigateur et système d'exploitation)
  • La date et l'heure du message
  • Le « loser » associé au message
Ces données sont visibles par les autres utilisateurs de la plateforme. Elles sont conservées pour toute la durée de vie du site.

Mesure d'audience

Ce site utilise Umami, un outil de mesure d'audience open source respectueux de la vie privée. Umami ne collecte aucune donnée personnelle, n'utilise pas de cookies, et les données sont hébergées sur nos propres serveurs (statistiques.nathanaelle.org). Aucun consentement n'est requis conformément aux recommandations de la CNIL.

Comptes utilisateurs

Les utilisateurs peuvent créer un compte sur Lostme afin de perdre.
Selon la méthode d'inscription choisie, les données suivantes sont collectées :

  • Inscription via Discord

    Mail, identifiant Discord, identifiant du channel privé bot, nom d'utilisateur Discord.

  • Inscription par email

    Adresse mail, nom d'utilisateur, mot de passe (stocké chiffré).

  • Inscription via Free

    Adresse mail, nom d'utilisateur, mot de passe (stocké chiffré), identifiant Free, clé API Free.
Ces données sont utilisées uniquement dans le cadre du fonctionnement du service et ne sont pas transmises à des tiers.

Base légale

Le traitement repose sur l'intérêt légitime de l'éditeur à assurer le bon fonctionnement du service et la traçabilité des échanges (article 6.1.f du RGPD).
